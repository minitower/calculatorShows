This site can work in 3 modes:

  1. Predict with known bid and campaign ID/name (one field)
  2. Predict with custom params (custom bid, approve, cr and ctr)
  3. Predict only shows for some campaign ID/name

For all this mode you can navigate by menu in left of screen. Calculator (now) run in development mode, so this app can't be protected to some cyberattack (for example, DDOS, MitM attack and so on)
